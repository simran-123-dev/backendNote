# 📥 Installing Node.js

Download Node.js from the official website:

https://nodejs.org

Node.js installation includes:

- Node.js Runtime
- npm (Node Package Manager)

---

# ✅ Verify Installation

Check Node.js version:

```bash
node -v
```

Example:

```bash
v22.0.0
```

Check npm version:

```bash
npm -v
```

---

# 📂 Create a New Project

Create a folder:

```bash
mkdir hello-node
```

Move into the folder:

```bash
cd hello-node
```

---

# 📦 Initialize npm

Create a package.json file:

```bash
npm init
```

or

```bash
npm init -y
```

`-y` creates the package with default values.

---

# 📄 package.json

`package.json` stores information about the project.

Example:

```json
{
  "name": "hello-node",
  "version": "1.0.0",
  "main": "index.js",
  "scripts": {
    "start": "node index.js"
  }
}
```

---

# 📌 Important Fields

| Field | Purpose |
|--------|----------|
| name | Project name |
| version | Current version |
| description | Project description |
| main | Entry point |
| scripts | Commands |
| author | Project author |
| license | License type |

---

# 📄 Create index.js

Create a file:

```
index.js
```

Example:

```javascript
console.log("Hello World");
```

---

# ▶️ Run the Program

Method 1:

```bash
node index.js
```

Output:

```
Hello World
```

---

# ▶️ Using npm Scripts

Add inside `package.json`:

```json
"scripts": {
    "start": "node index.js"
}
```

Run:

```bash
npm start
```

# 📁 Project Structure

```
hello-node/
│
├── index.js
├── package.json
└── package-lock.json
```

---

# 🔍 package-lock.json

This file:

- Stores exact package versions
- Keeps installations consistent
- Is automatically generated by npm

Do **not** edit it manually.

---

# 📦 What is npm?

npm stands for **Node Package Manager**.

Uses:

- Install packages
- Manage dependencies
- Run scripts
- Update packages

Example:

```bash
npm install express
```

---

# 💡 Difference

Running directly:

```bash
node index.js
```

Running with npm:

```bash
npm start
```

`npm start` executes the command defined inside `package.json`.

# 🎯 Key Takeaways

- Node.js executes JavaScript outside the browser.
- `npm init` creates the `package.json` file.
- `index.js` is commonly used as the entry point.
- `npm start` runs the command defined in the `scripts` section.
- `package-lock.json` ensures consistent dependency versions.
